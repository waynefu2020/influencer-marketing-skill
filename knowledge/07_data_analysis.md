# 数据分析复盘

## 核心理念

> 刚上手不要把数据分析想的很复杂。学会一个数据漏斗思路，后续无论做什么业务都会有思路。

做数据分析的目的：衡量博主合作的效果是否符合预期。

---

## 1. AARRR 漏斗模型

红人营销的数据漏斗，每一步都有关键指标：

```
内容曝光 → 内容互动 → 用户搜索 → 详情页/官网访问 → 加购物车 → 下单
```

| 阶段 | 关键指标 | 说明 |
|------|---------|------|
| 曝光 | 播放量、曝光量 | 内容触达了多少人 |
| 互动 | 点赞、收藏、评论、分享 | 内容是否引起兴趣 |
| 搜索 | 品牌词搜索量 | 用户是否主动搜索产品 |
| 访问 | 详情页 UV、官网流量 | 用户是否产生购买意向 |
| 加购 | 加购物车数 | 用户是否准备下单 |
| 转化 | 订单数、GMV | 最终成交 |

---

## 2. 如何衡量红人合作效果

### 核心观点

> 对博主合作来说，真正重要的不是 GMV 和 ROI，而是博主能否吸引更多人搜索你的产品，激发消费者行动起来。

- 真正负责 GMV 和 ROI 的是广告投放或直播带货（临门一脚）
- 要求一个博主直接完成曝光、搜索、转化三个阶段，是不现实的
- 直接要求红人营销的 GMV 是不对的

### 关键效果指标

1. **详情页访问量**：合作博主上线后，产品详情页的访问量是否有明显提升
2. **产品名称搜索量**：品牌词/产品词的搜索量是否增长

> 这两个指标有不错的提升幅度，就代表博主合作是有效果的。

## 3. 数据异常优化思路

### 通用分析框架

**定位问题 → 提出措施 → 优化 → 验证 → 巩固**

### 常见问题及分析思路

**问题 1：博主播放量很高，但没转化**
- 定位：漏斗在哪一步断了？可能是内容没有引导搜索/点击
- 措施：优化 Brief 中的 CTA（行动号召），加入搜索引导

**问题 2：博主平时播放量很高，发我们视频播放量很低**
- 定位：内容与博主日常风格不匹配，粉丝不感兴趣
- 措施：给博主更多创作自由度，让产品自然融入博主日常内容风格

**问题 3：视频点赞和收藏很少**
- 定位：内容互动阶段出问题，内容吸引力不足
- 措施：参考竞品爆款内容形式，调整内容策略

**问题 4：详情页访问多但下单少**
- 定位：转化阶段出问题，不是博主的问题，是产品详情页体验不好
- 措施：
  - 参考成熟竞品详情页重新设计
  - 设置打折、限时、新人优惠等活动，增加下单紧迫感

## 4. 复盘报告框架

### 单次 Campaign 复盘

```
1. Campaign 基本信息（时间、预算、目标）
2. 整体数据表现（曝光、互动、转化对比目标）
3. 各博主数据对比（按量级分类）
4. 爆款内容分析（什么内容表现好，为什么）
5. 数据异常分析（哪个环节有问题，原因是什么）
6. 优化建议（下一次怎么改进）
7. 预算使用情况（实际 vs 计划）
```

### 博主效果评估

```
- 博主名称 / 平台 / 粉丝量
- 合作形式 / 费用
- 内容形式 / 上线日期
- 播放量 / 互动量 / 互动率
- 详情页引流量（如有追踪）
- CPM / CPE（千人互动成本）
- 综合评级（是否值得复投）
```
